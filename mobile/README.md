# Simulai OAB - Mobile App

Aplicativo mobile para preparação da OAB usando React Native e Expo.

## 🚀 Tech Stack

- **Framework:** React Native 0.73 + Expo 50
- **Navigation:** React Navigation 6
- **UI:** NativeWind (TailwindCSS)
- **Animations:** React Native Reanimated
- **Auth:** Supabase Auth
- **Database:** Supabase (PostgreSQL)

## 🛠️ Desenvolvimento

```bash
# Instalar dependências
npm install

# Executar no simulador
npm start

# Executar no Android
npm run android

# Executar no iOS
npm run ios

# Build para produção
npm run build:android
npm run build:ios

# Linting e formatação
npm run lint
npm run format
```

## 📱 Features

- ✅ Banco de questões offline
- ✅ Simulados com timer
- ✅ Correção de peças por IA
- ✅ Plano de estudos sincronizado
- ✅ Vade Mecum offline
- ✅ Gamificação e progresso
- ✅ Sincronização com web

## 🏗️ Build & Deploy

- **Android:** Google Play Store via EAS Build
- **iOS:** App Store via EAS Build
- **Expo:** Over-the-air updates
