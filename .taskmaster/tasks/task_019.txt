# Description: Importar e processar o dataset de mais de 5.000 questões da FGV do Hugging Face, estruturando-o para uso eficiente no sistema.
# Details:
1. Desenvolver script Python para importação do dataset do Hugging Face usando datasets.load_dataset()
2. Processar e normalizar dados:
   - Classificar questões por disciplina, tema, ano, banca
   - Extrair artigos citados usando regex e NLP
   - Calcular nível de dificuldade baseado em estatísticas históricas

3. Implementar pipeline de ETL para atualização periódica
4. Criar índices de busca vetorial no PostgreSQL usando pgvector
5. Desenvolver sistema de versionamento para o dataset
6. Implementar validação e limpeza de dados
7. Criar metadados adicionais como palavras-chave e tópicos relacionados
8. Desenvolver API para acesso eficiente às questões
9. Implementar cache para questões frequentemente acessadas
10. Documentar estrutura e formato dos dados processados
